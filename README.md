# Театральная Студия — Telegram Bot

Telegram бот для операционного управления театральной студией. На первом этапе бот предоставляет быстрый доступ к ссылкам на внешние сервисы, с возможностью встроить функциональность позже.

## Установка

### Требования
- Python 3.9+
- pip

### Шаги установки

1. **Создайте виртуальное окружение:**
```bash
python -m venv venv
source venv/bin/activate  # на Windows: venv\Scripts\activate
```

2. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

3. **Создайте файл `.env`:**
```bash
cp .env.example .env
```

4. **Заполните переменные окружения в `.env`:**
   - `BOT_TOKEN` — получите у [@BotFather](https://t.me/botfather) в Telegram
   - `ADMIN_ID` — ваш Telegram ID (узнать можно у [@userinfobot](https://t.me/userinfobot))

## Запуск

```bash
python main.py
```

## Структура проекта

```
.
├── main.py                 # Точка входа
├── config.py              # Конфигурация
├── requirements.txt       # Зависимости
├── .env.example          # Пример переменных окружения
├── keyboards/            # Клавиатуры и inline кнопки
│   └── main_menu.py     # Главное меню
└── handlers/             # Обработчики команд и сообщений
    ├── __init__.py
    ├── commands.py       # Команды (/start, /help и т.д.)
    └── links.py         # Обработчик ссылок на сервисы
```

## Функциональность

### Текущие возможности
- Команда `/start` с главным меню
- Быстрый доступ к ссылкам на внешние сервисы
- Система логирования

### Планируемые возможности
- Встроенные формы для управления
- Интеграция с внешними API
- Уведомления и напоминания
- Управление расписанием

## Развертывание

Для работы в фоновом режиме можно использовать:
- `systemd` (Linux)
- `supervisor` (Linux/Windows)
- Cloud платформы (Heroku, Railway, VPS)
